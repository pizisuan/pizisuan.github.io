毕业旅行第五站：青海

应好友邀请到 青海 来了一次自驾游。这是一次相当棒的体验，我现在可以很明确的告诉你：自驾游 与 跟团游 完全不是一个东西。如果现在要我选择以后的旅行方式，我一定选 自驾游（条件允许的话），理由是：

  • 更自由（想去哪就去哪，行程完全自己定）
  • 更能锻炼人（攻略自己要提前弄好，要做好挺不容易的）

以后一定还要再来一次自驾游（独自一人）！！！

Example 1:

Input: [1,2,3]
Output: [1,2,4]
Explanation: The array represents the integer 123.

Input: [4,3,2,1]
Output: [4,3,2,2]

毕业旅行第四站：西安

五月份的最后一游，选择了西安。理由是：大学室友在这边可以带我飞！！！另外，西安个人觉得玩儿的还蛮多的。讲讲刚到西安给我的印象：古代与现代的结合体。古代 讲的是西安有非常多的“古代元素”：宫殿阁楼、城墙、寺庙、宝塔等等。现代 大家都懂得哈。住宿的地方我刚到周边，就发现街边马路好几群人围在一起：走近一看，在下象棋。难道西安人好下棋？想到刚从成都到这里，而成都人好麻将。每个地方的民风真的是不一样的，体会也自然不同，我想这也是旅行的意义所在。
