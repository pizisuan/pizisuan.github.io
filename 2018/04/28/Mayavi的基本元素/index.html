Mayavi库主要包括两大部分功能：

  1. MLAB模块：处理图形可视化和图形操作
  2. 其他：操作管线对象、窗口对象的API

MLAB模块

MLAB包含这样几类函数：绘图函数、图形控制函数、图形修饰函数、相机控制函数、其他函数、Mlab管线控制函数。

  • 绘图函数：实现已有数据进行可视化显示。数据可以使numpy数组构建的，也可以从外部数据读取的（如文件）。
  • 图形控制函数：对Mayavi中的figure进行控制。
  • 图形修饰函数：对当前绘制的图形进行修饰和一定的装饰。比如：坐标轴标识等。
  • 相机控制函数：对相机进行操作（移动、旋转等）。
  • 其他函数：对当前绘制图形赋予其他效果等。
  • Mlab管线控制函数：设置当前VTK中管线的数据源、增加数据集等。

类别 说明

API

Mayavi中的API可以分为两类对象：

  • 管线基础对象：用于管线操作
  • 主视窗和UI对象：用于视图绘制和UI绘制

主视窗和UI对象 MayaviScene、SceneEditor、EngineView
