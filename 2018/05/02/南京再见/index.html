南京再见

发表于 2018-05-02 | 更新于: 2018-05-02 | 分类于 旅游
字数统计: 592字 | 阅读时长 ≈ 2分钟

我不是复仇主义者，我无意于把日本军国主义欠下我们的血债写在日本人民的账上。
但是，我相信，忘记过去的苦难可能招致未来的灾祸。
梅汝璈

前天在南京逛了一天后，第二天（5月1号）上午去了侵华日军南京大屠杀遇难同胞纪念馆。天气阴沉，很应情。

现在已经回到上海了，接下来打算休息一天后再往下一站出发。趁着现在有时间，将南京的结尾部分写完。

侵华日军南京大屠杀遇难同胞纪念馆

一出地铁站，马路对面就有一座非常醒目的雕塑，看了之后让人觉得很压抑（真的太惨了）。我知道那就是侵华日军南京大屠杀遇难同胞纪念馆所在地了。

雕塑长墙的起首，是一尊高达11米的母子雕塑，这便是主题雕塑《家破人亡》，底座上镌刻着：“被杀害 儿子永不再生；被活埋 丈夫永不再生；悲苦留给了被恶魔强暴了的妻子，苍天啊……”抱着孩子萎谢的身子，受辱的母亲眼泪也已干涸……

我知道，我看到的这座雕塑只是南京大屠杀时期的众生相之一罢了，还有更多悲惨的。我想每个中国人站在这些雕像前的心情都会很沉痛吧。由于行程上的安排，没有进入馆内参观，但是不难想象里面会有怎样的一些场景。我只想说：历史需要铭记，我们今天的生活来之不易！

总结

其实当我看到纪念馆的时候，一方面觉得心情很沉重，另一方面又觉得我们国家充满了希望。沉重 是因为想到当年那些遇难同胞的悲惨；希望 是因为看到来参观的人很多很多，尤其是看到很多大人都会对自己的小孩讲解这段历史的时候，我觉得中国老百姓是有心肝的，我们的民族是有希望的。

最后这里附上电视剧《血色湘西》中的一段经典对白。

再见，南京！！！好吧，收拾好心情准备出发下一站！未完待续。
